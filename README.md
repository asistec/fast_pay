Yahoo! FastPay sample code
==============================


Live demo
------------------

- See
  - http://matsu.teraren.com/static/fast_pay/

Requirements
------------------
- bower
- PHP 5.3=<
- composer


Setup
-------------------

```
% git clone git@github.com:matsubo/fast_pay.git
% cd fast_pay
% bower install
% php composer.phar install
% echo 'YOUR SECRET KEY' > secret.key
```
